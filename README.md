# My RecipePage Library

A React Redux website for storing recipes

Built with:
* Golang
* Typescript
* React/Redux
* Mysql
* Redis

# Local Development
## Requirements
* Docker
* Go 1.12+
* [Watchtower](https://github.com/iplay88keys/watchtower) installed on PATH

## Running
```bash
go mod download

./scripts/run.sh
```

# Running the tests
```bash
./scripts/test.sh
```
