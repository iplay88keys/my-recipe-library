# My RecipePage Library

A React Redux website for storing recipes

Built with:
* Golang
* Typescript
* React/Redux
* Mysql
* Redis

# Local Development
## Requirements
* Podman
* Go 1.25+
* Node
* Yarn
* React
* Mysql
* Flyway

## Running
```bash
go mod download
yarn --cwd "ui" install

./scripts/run.sh
```

# Running the tests
```bash
./scripts/test.sh
```
